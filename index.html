This is a generic Wiki-table simulator. It does not manually hardcode boss/NPC loot tables. It approximates one action as guaranteed rows + a configurable number of weighted main rolls + independent special checks. Exact mechanics like raid points, invocation scaling, rare-drop-table nesting, diary modifiers, combat achievements and other hidden formulas may need custom logic later.
